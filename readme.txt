ConvertRAWtoNETCDF

Author: 
Sindre Vatnehol
Institute of Marine Research, Norway
sindre.vatnehol@gmail.com


Purpose: 
The purpose of this script is to enable conversion of sonar .raw to .nc


Requirement: 
The project has been tested using Python 3.5. 
Pachages such as numpy, netCDF4,  must be install prior to initialisation. 


Status: 
Testing of the project is almost finished. 


Future updates:

    - Add other sonar/echo sounder equipment
